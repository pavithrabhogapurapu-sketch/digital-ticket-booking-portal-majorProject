# digital-ticket-booking-portal-majorProject
## 📌 Overview

Digital Ticket Booking Portal is a simple front-end web application developed using HTML, CSS, and JavaScript. It allows users to view available movies and simulate ticket bookings through an interactive interface. The application also includes a basic management panel for adding movie show details.

## 🎯 Aim

To develop a responsive digital movie ticket booking portal that provides an easy-to-use interface for booking movie tickets and managing movie show information.

## ✨ Features

- Modern and responsive user interface
- Display available movie shows
- Movie details including screen, timing, and ticket price
- One-click ticket booking confirmation
- Admin panel to add movie show details
- Attractive Slate Grey and Purple theme
- Built using pure HTML, CSS, and JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open the `index.html` file.
4. Right-click the file and select **Open with Live Server**.
5. The application will open in your default browser.
